# m365-security-hardening-lab
**Microsoft 365 Security Hardening Lab**

This repository provides a structured, step-by-step lab for securing a Microsoft 365 tenant using real-world best practices. It is designed for security engineers, consultants, and learners who want practical experience with identity hardening, email security, endpoint protection, DLP, and monitoring configurations.

The lab environment mirrors how modern SMEs and enterprises configure security in production, with guidance on Conditional Access, Defender AV baselines, Safe Links/Safe Attachments, sensitivity labels, and alerting. Each section includes screenshots, examples, and recommended configuration settings.
